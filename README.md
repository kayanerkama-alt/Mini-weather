## 🚀 Getting Started

### Online (No Setup)
```bash
# Simply clone and open
git clone https://github.com/kayan4bit/mini-weather.git
cd mini-weather
open index.html
```

### Local Development
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve

# PHP
php -S localhost:8000
```

Visit: `http://localhost:8000`

## 🎯 How the Garden Works

The interactive garden responds to weather conditions in real-time:

**🌻 THRIVING (Score 90+)**
- Perfect temperature (15-25°C)
- Ideal humidity (40-70%)
- Calm winds (<15 km/h)
- Light rain
- Safe UV (<3)

**🌱 HEALTHY (Score 70-89)**
- Good temperature range
- Acceptable humidity
- Gentle breezes
- Moderate conditions

**🌾 STRESSED (Score 50-69)**
- Less ideal conditions
- Extreme temperatures
- Poor humidity
- Strong winds

**🍂 WILTED (Score <50)**
- Harsh conditions
- Freezing or extreme heat
- Very low/high humidity
- Severe weather

## 🐛 Troubleshooting

**Location not working**
- Enable location services in browser settings
- Allow permission when prompted
- Try refreshing

**No weather data**
- Check internet connection
- Verify location services are on
- Try different browser
- Clear cache

**Notifications not showing**
- Ensure browser supports notifications
- Check notification permissions
- Restart browser
- Try clicking Alerts button again
